# Requirements

1. Pygame, How to install:

	```
	pip install pygame

	````
2. Tkinter, How to install:

	Ubuntu:
	```
	sudo apt-get install python3-tk
	```
	windows: 
	
	https://tkdocs.com/tutorial/install.html#install-win-python

	Mac:
	
	https://tkdocs.com/tutorial/install.html#install-mac-python

# How to Run
```
python app.py
```
