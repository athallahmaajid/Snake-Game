# Requirements

1. Pygame, How to install:

	```
	pip install pygame
	````
2. Tkinter, How to install:

	Ubuntu:
	```
	sudo apt-get install python3-tk
	```
	Mac:
	
	https://tkdocs.com/tutorial/install.html#install-mac-python
3. Pillow, How to Install:
	```
	pip install pillow
	```

# How to Run
```
python app.py
```
